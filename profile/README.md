#### 🛠️ Security Skills & Slash Commands
 
| Skill | Command | Description |
|---|---|---|
| **Pentest** | `/pentest` | 7-phase PTES methodology. Deploys 35+ agents in parallel. CVSS 3.1 reports with screenshots, videos, and HTTP log evidence. |
| **HackerOne** | `/hackerone` | Parse program scope, run parallel tests across all in-scope assets, validate PoC, output platform-ready HackerOne / Bugcrowd submissions. |
| **CVE Testing** | `/cve-testing` | Identify technology stack, research matching CVEs, run automated exploitation attempts, validate with proof-of-concept. |
| **Domain Assessment** | `/domain-assessment` | Subdomain discovery, port scanning, service detection, and attack surface inventory generation. |
| **Web App Mapping** | `/web-application-mapping` | Passive + active endpoint discovery, technology detection, headless browser automation via Playwright. |
| **AppSec Patterns** | `/common-appsec-patterns` | Quick-hit OWASP Top 10 testing — XSS, SQLi, SSRF, auth flaws, and security misconfigurations. |
| **Authenticating** | `/authenticating` | Auth workflow testing, 2FA/MFA bypass, CAPTCHA solving, bot detection evasion, OTP handling. |
